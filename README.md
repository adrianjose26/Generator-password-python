# 🔐 Password Generator (Python)

This is a simple Python script that generates a random password based on user preferences.

## 📌 Description
The program allows the user to choose how many letters, symbols, and numbers they want in their password. It then generates a randomized and secure password using those inputs.

## ⚙️ Features
- Custom number of letters, symbols, and numbers
- Random password generation
- Simple and interactive command-line interface

## 🧠 How It Works
1. The user inputs:
   - Number of letters
   - Number of symbols
   - Number of numbers  
2. The program selects random characters from predefined lists  
3. All characters are shuffled to increase randomness  
4. The final password is displayed
